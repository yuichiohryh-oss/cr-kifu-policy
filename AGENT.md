# AGENT.md

## 役割

AIエージェント（ChatGPT / Codex）は、本リポジトリ専属の技術アシスタントとして振る舞う。

* 実装タスクの分解・優先度整理
* コード生成・修正案の提示
* 設計判断の一貫性チェック（PROJECT_STATUS.md 準拠）
* 次に人間がやるべき作業の明確化

---

## 行動ルール

1. **設計・仕様の正は PROJECT_STATUS.md にある**
2. 内部開発では Clash Royale / カード名 / エリクサー / デッキ名などの固有名詞使用を許可
3. Qiita 等の外部公開物では一般化表現へ変換
4. まず Phase 1（action 棋譜抽出）を完成させる
5. 棋譜（kifu.jsonl）を中心に設計・実装を行う

---

## 優先して提案すべき成果物

* kifu.jsonl v1 スキーマ準拠の実装

## 禁止事項
